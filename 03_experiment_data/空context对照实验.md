# 空 context 对照实验

## 实验设计

使用 Hermes 14B 模型，控制变量：
- 对照组：空 context 直接回答问题
- 实验组：使用 zhongxing 管道压缩后的 context 回答问题

## 关键发现

### 对照组（空 context 直接答）
- **严重幻觉**：编造"美杜莎是希腊神话""萧曼陀（萧炎父亲）""废灵体"等完全不存在的内容
- 速度：9s/题
- 答案完全不可用

### 实验组（管道压缩 context）
- 所有输出严格限定在上下文范围内
- 不会编造离谱内容
- Q2（萧战）、Q5（厄难毒体）完全正确
- 速度：14s/题

## 核心结论

1. **准确率提升来自压缩检索，不是模型自带知识**
2. 管道多花 50% 时间，换来的是从"随机瞎编"到"可追溯推理"的质变
3. 无管道时 14B 纯靠幻觉——知识面广但不可控
4. 有管道时 14B 严格基于上下文——知识面窄但可信

## 对 v2.0 的意义

空 context 实验验证了架构的**核心价值**：
- 压缩管道不是可选项，而是让模型从"瞎编"到"可信"的必需组件
- v2.0 即使面向 V4-Pro 级大模型，压缩管道仍是不可或缺的前置环节
