# 配置文件说明

## 模型配置

```python
FALLBACK_CONFIG = {
    "extractor_model": "qwen2.5:1.5b",       # 通用提取器（3个并行）
    "schema_model": "qwen2.5:3b",             # 维度自发现 + NER
    "big_brain_model": "qwen2.5:14b",         # 推理
    "num_extractors": 3,
    "max_feedback_rounds": 2,
}
```

## 关键参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| num_extractors | 3 | 并行提取器数量 |
| max_feedback_rounds | 2 | 反馈环最大轮数 |
| chunk_size | 900 | 分块大小（字） |
| RRF_K | 60 | RRF 排序常数 |
| timeout | 180s | Ollama API 超时 |
