# R 字段 Grounding 方案

## 问题背景

R 字段（关系字段）在提取中容易出现以下问题：
1. **关系丢失** — 共现实体间的关系关键词（如"朋友"）未被提取
2. **关系串门** — 关系安错实体
3. **虚假关系** — 提取器编造不存在的关系

## 方案

### 1. 确定性共现扫描（已实现）

在分句窗口内检测共现实体间的预定义关系关键词：

```python
rel_patterns = {
    "朋友": "朋友", "师徒": "师徒", "敌人": "敌人",
    "对手": "对手", "弟子": "弟子", "师父": "师父",
    "徒弟": "徒弟", "盟友": "盟友", "叛徒": "叛徒",
}
```

- 先按句号/感叹号/换行切分
- 再按逗号/分号细分（缩小共现窗口）
- 给离关键词最近的 2 个实体加关系

### 2. 3b NER 补漏（已实现）

- 3b 列出原文所有人名
- 与已提取实体做模糊匹配
- 漏掉的补回

### 3. v1.8.4 增强方案（待实现）

- 增加更丰富的关系词库
- 加入关系方向检测（A→B 与 B→A 不同）
- 支持跨句关系推断（仅限于同一段落内）

### 4. 未来方案

- 语义检索替换关键词匹配
- 利用 3b 推理能力做关系二元组验证
- 引入关系置信度评分

## 当前限制

- 仅支持预定义关系词（有限集）
- 跨段关系丢失
- 无关系方向标注
