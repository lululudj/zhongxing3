# 紧凑化规则集

## 概述

30+ 条通用确定性规则，覆盖 8 大类冗余模式。不调用模型，纯字符串处理。

## 8 大类规则

### 1. 虚词/判断词删除

删除不改变语义的判断性词语：

```
乃是 → 空
则是 → 空
即为 → 空
亦为 → 空
便是 → 空
所谓 → 空
拥有 → 空
正是 → 空
不愧是 → 空
算得上 → 空
```

### 4. 等级模式压缩

```

```

例：`斗皇级别的强者` → `斗皇`

### 5. 因果链压缩

```

### 8. 尾缀清理

```

## 规则设计原则

1. **确定性** — 不调用模型，纯字符串/正则处理
2. **语义不变** — 只删除不改变事实的成分
3. **通用性** — 适用于小说/论文/法律/财报等各类型文本
4. **可叠加** — 多条规则可串联应用
